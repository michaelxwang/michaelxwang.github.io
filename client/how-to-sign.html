How to sign

I have uploaded a "*_SIGNED PDF file for you to sign, you can use one of the following alternative ways to sign.

  • Print the form, sign and date, scan, and upload the image.
  • Sign the PDF file on Apple computer following this instruction.
  • Sign the PDF file on Windows computer following this instruction.

Please handwrite your signature instead of using a printed font, and also rename the file to *_SIGNED when you upload.
